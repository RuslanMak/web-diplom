## Start

1. composer instal
2. php artisan storage:link
3. cp .env.example .env
4. php artisan key:generate
5. php artisan config:cache
6. php artisan migrate

For changing
1. npm install
2. npm run watch-poll

## NB
in db->users
role = 1 --- admin
role = 2 --- controller
